Curso: How to Design Programs

Diseña programas de forma estructurada, lógica y comprensible

Introducción

Este curso sigue el enfoque del libro How to Design Programs (HTDP). Aprenderás a desarrollar software aplicando una receta sistemática que comienza con datos simples y crece hacia la abstracción y el diseño funcional.

Contenido del Curso

I. Fixed-Size Data

Introduce cómo diseñar programas para datos simples y estructuras fijas: números, cadenas y registros. Aquí se aplican las primeras plantillas de diseño.

Leer sección

II. Arbitrarily Large Data

Aprende a manejar estructuras de datos de tamaño indefinido, como listas y árboles, usando recursión estructural y plantillas recursivas.

Leer sección

III. Abstraction

Una vez que has visto varios patrones, esta sección muestra cómo abstraer el comportamiento común usando funciones de orden superior.
